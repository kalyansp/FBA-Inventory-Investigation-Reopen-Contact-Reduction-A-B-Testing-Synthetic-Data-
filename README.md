# FBA-Inventory-Investigation-Reopen-Contact-Reduction-A-B-Testing-Synthetic-Data-
We aim to build a AB test model that replicate the Amazon seller issue scenario with the synthetic data

This project was built as a portfolio demonstration of end-to-end A/B test design, 
synthetic data generation, exploratory data analysis, and statistical hypothesis testing 
applied to a real-world Amazon FBA operations context.

We want to determine if the observed reduction in case reopen rate between the control and treatment group is statistically significant.

Disclaimer:
This project is purely pedagogical purpose to demonstrate my knowledge on Statistical topics and data used is synthetic/fabricated data no real world data is used to maintain confidentiality.
Recreated the real-world problem of FBA short-shipment misclassification using a synthetically generated 
dataset replicating seller inventory, shipment, and investigation attributes
